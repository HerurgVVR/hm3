  • Download and extract the program somewhere
  • Edit the hm3.ini file to your liking. Setup the storage directories and DB configuration for your needs. See the comments in the ini file for more information.
  • Run "php ./scripts/config_gen.php" from the main directory to generate the production site. If you change the hm3.ini file, you need to re-run this command for your changes to take effect.
  • Symlink or move the newly created files in the "site" directory to the web-server where you want to host the software.
  • You can enable a debug site by symlinking or moving the main download directory to the web-server document root and accessing it's index.php file. Don't do this in production. It could be bad.
