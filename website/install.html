Requirements

Cypht requires at least PHP 5.4, with the OpenSSL and cURL extensions. You will also need PDO support if using any databases features. Testing is done on Debian and Ubuntu platforms with Nginx, Apache, standard PHP, and HHVM.

Steps

  • Download and extract the program somewhere
  • Copy the hm3.sample.ini file to "hm3.ini" and edit to your liking. Setup the storage directories and DB configuration for your needs. See the comments in the ini file for more information.
  • Run "php ./scripts/config_gen.php" from the main directory to generate the production site. If you change the hm3.ini file, you need to re-run this command for your changes to take effect.
  • Sym-link or move the newly created files in the "site" directory to the web-server where you want to host the software.
  • You can enable a debug site by sym-linking or moving the main download directory to the web-server document root and accessing it's index.php file. DON'T DO THIS IN PRODUCTION. IT WOULD BE BAD.
  • If you are using the DB option for user authentication, you can create and delete users, as well as change their passwords, with the commands in the scripts/ directory
