All your messages, from all your accounts, in one place. Cypht is like a news reader, but for E-mail. Cypht does not replace your existing E-mail accounts - it combines them into one. And it's also a news reader :).

Cypht is an Open Source webmail application written in PHP and JavaScript that speaks native IMAP, POP3, and SMTP. This is a third generation E-mail client from the developers of the Hastymail project.

Cypht has an evolving feature set, and employs a modular design that provides an unprecedented level of site specific customization options. The code was built to aggregate data, and while initial development is primarily concerned with E-mail, it could be used for any type of data source.

Like the efforts that preceded it, the focus of this project is security, speed, and simplicity. In that order. Cypht is lightweight in the browser, on the server, and through the network.

If you are looking for a webmail solution for an ISP, school, business, or just personal use, I hope that Cypht can help.
